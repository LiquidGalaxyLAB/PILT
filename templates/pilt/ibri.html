Interactive Beacon Rescue Interface

This project is proposing of a civil security solution using physical web as main technology and drones (UAV) to find missing people.

The way to find them is by using multiple unattended aerial vehicles (as known as drone).

The main idea from this project is to find a missing person scanning an area with one or more drones who will be receiving the physical web beacon Bluetooth signal to determinate the gps position from it in case of emergency.
